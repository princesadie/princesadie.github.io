Exploring CSS

December 16, 2015

This is the third week of Dev Bootcamp and we focused on expanding our knowledge of CSS. To start off with, CSS stands for Cascading Style Sheets and it describes how a page or document on the web should be displayed. CSS is pretty awesome because it cuts a lot of the grunt work with adding styles since it can be applied to multiple pages at once. It wasn't a difficult subject to learn, but positioning for me was like learning how to skateboard for the first time - right when I thought I had it, I slip and take a fall.

  • What is the difference between margin, border, and padding?

    • The margin is the outermost and the space in-between elements. The border is the line in between the margin and the padding. The padding is the inner most space next to the element and behind the border - it comprises of the space between the element itself and the border.

  • What are the best practices associated with using classes vs. ids?

    • If there are multiple elements that require the same styling, then classes should be used, otherwise an ID that's specific to one particular element can be used.
